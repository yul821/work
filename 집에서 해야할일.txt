집에서 해야할 리스트
1. 깃다운 설치후
2. 깃 허브 폴더 만들고 : 비주얼 스튜디오코드에서 열기 
3. 깃 환경설정 : 교안 깃 기본 파트에서 1번 작업
4. 깃 초기화 : 교안 깃 기본파트에서 2번 작업 


5. 깃 연결하기 교안 깃 허브 1번 작업
 ex) 비주얼 스튜디오 코드에서 (폴더)> git remote remove origin (깃 허브
에서 복사한 리포지토리 주소 쓰기 엔터

6. 깃 허브 연결 확인하기
git remote -v 엔터

7. 원격저장소(리포지토리)이 파일 내려	받기 git pull origin master
=> 폴더 생성됨 -> 비주얼 스튜디오 코드에서 생성된 폴더 열기 : file - open
folder 
8. 오늘 작업한 파일 3개 수정 : 2라인 이상 추가하기 !
-> 파일 저장 -> 스테이징 -> 커밋 -> 푸시하기(git push -u origin master)
